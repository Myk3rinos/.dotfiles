Dot Files for NixOS

Includes:
- `configuration.nix` for NixOS 
- `neovim`
   - neotree
   - lualine
   - gitCopilot
   - lsp support
   - tree-sitter
   - telescope
   - fzf
- `yazi`
- `kitty`
- `zsh`
- `starship`
- `neofetch`
- `theme` for gnome

## Installation

clone the repo to your home directory

execute the `install.sh` script
